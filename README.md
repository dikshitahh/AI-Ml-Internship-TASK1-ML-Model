# AI-Ml-Internship-TASK1-ML-Model

#Machine Learning Model Implementation

##Overview
This project implements a machine learning classification model using the iris dataset. Logistic Regression was used to classify iris flowers into Three species based on flower measurements.

##Dataset
- Iris Dataset
- 150 samples
- 4 features
- 3 classes

#Technologies Used
- Python
- Pandas
- Numpy
- Scikit-learn
- Matplotlib
- Seaborn

- Model Training
- Model Evaluation
